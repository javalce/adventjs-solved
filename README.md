## Quickstart

- Instalar

```bash
pnpm install
```

- Ejecutar los retos

```bash
pnpm test
```

- Ejecutar un reto en concreto

```bash
pnpm test YYYY/reto-XX # Ejemplo: pnpm test 2023/reto-01
```
